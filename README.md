# VitaDeck

[![npm version](https://img.shields.io/npm/v/%40vitadeck%2Fsdk.svg)](https://www.npmjs.com/package/@vitadeck/sdk)

VitaDeck lets you build Deck Apps for PlayStation Vita.

Install on your Vita:

- Vita VPK: [GitHub Releases](https://github.com/matemolnar8/vitadeck/releases)

Start building an app with the SDK:

- SDK docs: [`js/packages/sdk`](js/packages/sdk)
- npm package: [`@vitadeck/sdk`](https://www.npmjs.com/package/@vitadeck/sdk)

```bash
npx @vitadeck/sdk create my-deck-app
```

Agent skills (Cursor, Claude Code, etc.):

```bash
npx skills add matemolnar8/vitadeck
```
